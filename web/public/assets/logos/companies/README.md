# Company logos

One default SVG for each distinct company currently named in the live provider
registry or the planned AI and object-storage categories.

Website URLs use:

```text
/assets/logos/companies/<file>
```

`manifest.json` maps every label used by the site to its parent company, records
where the asset came from, and notes the surface the supplied variant is intended
for. Product labels such as AWS S3, Cloudflare R2, and Vercel Sandbox deliberately
reuse their parent-company mark.

The Tigris source asset is a white wordmark designed for a dark surface. OpenAI,
Anthropic, Vercel, and Fireworks use dark artwork intended for a light surface.
Keep these original files unchanged; add an official alternate variant rather
than recoloring or editing a mark.

Company names and logos are trademarks of their respective owners. Review each
owner's current brand and trademark guidance before public use. The Simple Icons
files are community-maintained reproductions; their source URLs are recorded in
the manifest so they can be replaced with official packages when available.
